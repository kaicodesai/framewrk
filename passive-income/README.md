# Passive Income Project: Agent Ops Starter Kit

A digital product, not a service — a one-time-built pack sold as a download, with no ongoing
delivery, outreach, or ad spend required to keep it "running." This is deliberately different
from `framewrk/` (the sibling business in this repo), which depends on continuous founder-led
outreach to sell websites; this project is meant to sell via marketplace/search discovery
instead.

See `claude-code-starter-kit/` for the actual product: a pack of Claude Code skills, subagents,
and hooks, plus the Gumroad listing copy and license terms needed to sell it.

## Why this satisfies "no marketing required"

- Distribution is Gumroad's own search + tag system, not paid ads.
- The realistic path to the first sales still needs **one** seed post in a relevant community
  (see `claude-code-starter-kit/SALES_PAGE.md`'s pre-launch checklist) — that's a single
  one-time action, not a recurring marketing campaign.
- After that, it's genuinely passive: no inventory, no support obligation beyond what the
  license promises, no server to run.

## Launch steps (manual, can't be automated from here)

1. Read every file in `claude-code-starter-kit/` once more yourself before publishing.
2. Create a Gumroad account/product listing, paste in `SALES_PAGE.md`'s copy, set the price.
3. Run `claude-code-starter-kit/package.sh` to build the zip and upload it as the product file.
4. Publish, then do the one seed post from the pre-launch checklist.

## Honest expectations

This is a real, sellable, finished product — not a guaranteed income stream. Digital info/config
products in a specific technical niche typically see modest, inconsistent sales unless the
niche has real search volume and the seed post lands somewhere the right audience actually
looks. Treat the first week as a test of the price and positioning, not a verdict on the whole
idea — cheap to iterate on (re-price, tweak the listing, add a second bundle) since there's no
sunk infrastructure cost.
