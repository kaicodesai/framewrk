# Framewrk

Automated business website build pipeline: Cloudflare Worker backend
(`framewrk/`), operator dashboard (`framewrk-dashboard/`), and the business
blueprint/build spec that define how it works (`docs/`).

Each client website lives on its own branch off `main` (e.g.
`clarity-fitness`), separate from the pipeline/dashboard code.

Migrated from `kaicodesai/business-agent-foundry` (branch
`claude/repo-agent-overview-mr9jqo`) as a clean snapshot — see
`framewrk/README.md` and `docs/specs/framewrk-build-spec.md` for details.
